### ファイルのステータス確認
どのファイルがコミット対象か確認できる。

``` git
$git status
```

### ファイルをステージに追加する
※コミット前にインデックスに追加する。インデックスに追加しないとコミットできない。
``` git
$git add
```

### コミット
``` git
$git commit
```
* Windows でコマンドラインでコミットする場合はコミットメッセージ入力後に下記を入力する。
```git
:wq
```

### ファイルを移動する
``` git
$git mv [移動前パス] [移動後パス]
```
この時に、[git mv]コマンドを入力すると物理ファイルも自動で移動してくれるので、カット&ペースト等で移動させる必要はない。
