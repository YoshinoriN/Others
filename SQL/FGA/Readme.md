#### 全テーブルにFGAを張るサンプル

##### このサンプルを作った背景
--------------------
* どのテーブルがどのようなSQLで更新されたか横着して確認したい。

##### ちなみに
--------------------
* 全テーブルに張るだけなら、[標準監査]で対応できる。(今回のようにFGAはつかわなくても使わなくてもよい・・・・)
* 本番環境での使用厳禁 !!
* 今回のサンプルの場合、監査ポリシーはテーブル名で作成される。
