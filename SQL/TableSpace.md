#### 表領域の使用量を表示する
```sql
SELECT
	T.TABLESPACE_NAME 表領域名,
	TRUNC(T.BYTES/(1024*1024),0) 総容量,
	ROUND((T.BYTES - SUM(F.BYTES)) / (1024*1024),0) 使用容量,
	ROUND(SUM(F.BYTES) / (1024*1024),0) 空き容量,
	ROUND((1-SUM(F.BYTES) / T.BYTES)*100,2) 使用率
FROM
	SYS.DBA_FREE_SPACE F,
	(SELECT TABLESPACE_NAME,
		SUM(BYTES) BYTES
	FROM SYS.DBA_DATA_FILES
	GROUP BY TABLESPACE_NAME) T
WHERE T.TABLESPACE_NAME = F.TABLESPACE_NAME(+)
GROUP BY T.TABLESPACE_NAME, T.BYTES;
```

#### 自動拡張確認
```sql
SELECT
	 TABLESPACE_NAME
	,FILE_NAME
	,AUTOEXTENSIBLE
FROM
	 DBA_DATA_FILES;
```
